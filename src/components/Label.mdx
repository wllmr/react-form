# Label

A component that handles label for inputs, more specific the following:

- stylings
- behaviour
- a11y

### Basic example

```tsx
export type LabelProps = JSX.IntrinsicElements['label'];

export const Label = ({ ...props }: LabelProps) => {
  return <label {...props} />;
};
```
